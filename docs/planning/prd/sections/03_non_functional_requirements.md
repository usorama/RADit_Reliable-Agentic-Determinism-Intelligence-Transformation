# 03. Non-Functional Requirements (NFR)

-   **NFR-01 Latency**: Agent response for simple tasks < 5s. Complex planning < 60s.
-   **NFR-02 Cost Control**: Admin dashboard to set daily token budget caps.
-   **NFR-03 Data Privacy**: "Confidential Mode" where no code leaves the VPC (using local LLMs or Zero-Data-Retention APIs).
-   **NFR-04 Traceability**: Full transparency logs ("Chain of Thought") for every decision made by the agent.
